HTML

  • HTML is used to create the structure of a webpage
  • HTML is made up of elements which work together to provide content and the basic framework for all pages on the web
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The elements seen on a page include text, image, list and headers
  • The elements used to divide and describe a page are meta-related, content division and semantic
  • lang-en tells what type of language will be used for webpage
  • lang-en is an html attribute
  • Attributes define additional info about an element
  • HTML attributes are always defined in the beginning element
  • Metadata is data about the webpage that the browser uses to effectively serve our webpage to users
  • Metadata is not seen by users and contains important info about accessibility, search engines and performance

CSS

  • Cascading Style Sheets (CSS) is a programming language used by developers to define how webpage content is presented to users. It defines the colors, aesthetics, visual layout and organization
  • A selector defines the element or attributes to which the rules or declarations apply
  • Declarations contain two important components: The CSS property we want to apply and the value of the property
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When we create an external CSS file, we need to link it to the HTML file using the "link" element

Git

  • Git is a version control system that let's you manage and keep track of each version of your project
  • Cit commands always start with "git"
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • GitHub is a cloud-based hosting service that allows you to keep track of projects, store code and collaborate with others
  • repository: any project in GitHub
  • branch: wworkspace where you can make changes that don't affect the live site
  • markdown: the way content files are formatted in GitHub
  • commit changes: saving a new version of a file
  • pull request: asking for changes on a branch to merge into another branch
  • merge: pull request is approved and pulled into main branch
  • issue: how work is tracked and a way to report new tasks or content fixes

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • In JavaScript, to declare a variable, we use the "var" keyword. After the "var" keyword, we give the variable a name.
  • A variable is a named container that allows us to store data in our code.
  • This name must be unique because it is how we will reference the information stored in the variable in our code.
  • We use an assignment operator "=" to assign the specific data that we want stored in our variable.
  • The "topics" variable contains a "string" that includes all of the topics that we are studying in this module
  • The word "console" refers to a test environment that developers use to check out their code
  • The second part,".log()", is a method, which means a set of instructions that can be executed by a computer.
  • We can use the "console.log" method to output a message to the web console by adding an argument
  • Control flow is the order in which a computer executes code in a script.
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page
  • Control flow can be interrupted, however. We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • When the computer encounters an "if" statement in a JavaScript file, it will check to see if the condition that we provide is truthy
  • If it is, then it will do whatever action we ask it to do in between the curly brackets
  • If the condition isnt true or has a value that isnt truthy, then the computer will ignore whatever is in between the curly brackets and move on
  • In JavaScript, we can have more than one condition by using “else if”
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way
